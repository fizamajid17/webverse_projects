Bank Account Mini-System

A simple Python console program that uses functions, state management, and loops.

Features

Deposit money

Withdraw money (with insufficient balance check)

Check balance

Menu repeats until the user exits

How the Program Works

balance starts at 0

deposit(amount) → adds money

withdraw(amount) → checks and removes money

check_balance() → prints the current balance

A while True loop repeatedly shows the menu
